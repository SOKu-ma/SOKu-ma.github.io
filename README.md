# SO-KUMA Labs Pages

SO-KUMA Labs の公式サイトを公開するための GitHub Pages リポジトリです。

公開URL:

- [https://www.so-kuma.com](https://www.so-kuma.com)

## Structure

- `index.html`: トップページ
- `privacy/index.html`: プライバシーポリシー
- `support/index.html`: サポートページ
- `styles.css`: 共通スタイル
- `favicon.svg`: サイトアイコン

## Deployment

このリポジトリは GitHub Pages の `main` ブランチ `/ (root)` から配信する前提です。
